type evmc_get_capabilities_fn = Option<unsafe extern "C" fn(*mut evmc_instance) -> u32>;

Return the supported capabilities of the VM instance.

This function MAY be invoked multiple times for a single VM instance, and its value MAY be influenced by calls to evmc_instance::set_option.

@param instance The EVM instance. @return The supported capabilities of the VM. @see evmc_capabilities.
